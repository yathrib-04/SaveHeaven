# Save Heaven

Safe. Supportive. Secure.

Save Heaven is a full-stack web platform that provides a safe space for reporting incidents, accessing support resources, and connecting users with help. It is designed with a strong focus on confidentiality, security, and compassionate support.

Save Heaven allows users to securely submit incident reports with evidence, explore curated support resources, and manage their activity through a personal dashboard. The platform is simple, private, and fully responsive across devices.

Features include secure incident reporting with file uploads (images, video, and audio), a resource library, user dashboard, feedback submission, authentication, profile management, and a modern responsive user interface.

Tech Stack:
Frontend: React, Vite, Tailwind CSS, React Router
Backend: Node.js, Express, MongoDB, Mongoose, Multer, JWT authentication

Getting Started:
Prerequisites: Node.js, MongoDB, Git

Setup:
git clone <repository-url>
cd Save-Heaven

Backend:
cd Backend
npm install
npm start

Frontend:
cd Frontend
npm install
npm run dev

Security & Privacy:
Save Heaven uses secure authentication, protected routes, safe file upload handling, environment-based configuration for sensitive data, and follows a privacy-first design approach.

Purpose & Impact:
Save Heaven aims to make incident reporting safer, reduce barriers to seeking help, and provide access to essential support resources in a respectful and supportive digital environment.
